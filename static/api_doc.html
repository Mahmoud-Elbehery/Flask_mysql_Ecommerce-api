API Documentation

Authentication Endpoints

POST /api/auth/admin/authenticate

Authenticate an admin user and get a JWT token.

Request Body:
{
  "email": "admin@example.com",
  "password": "password"
}

Response:
{
  "token": "your_jwt_token"
}

User Endpoints

GET /api/users

Retrieve all users.

Response:
[
  {
    "id": "user_id",
    "email": "user@example.com",
    "first_name": "First",
    "last_name": "Last"
  }
]

GET /api/users/<user_id>

Retrieve a specific user by ID.

Response:
{
  "id": "user_id",
  "email": "user@example.com",
  "first_name": "First",
  "last_name": "Last"
}

POST /api/users

Create a new user.

Request Body:
{
  "email": "newuser@example.com",
  "password": "password",
  "first_name": "First",
  "last_name": "Last"
}

Response:
{
  "id": "new_user_id",
  "email": "newuser@example.com",
  "first_name": "First",
  "last_name": "Last"
}

PUT /api/users/<user_id>

Update a user.

Request Body:
{
  "first_name": "Updated",
  "last_name": "Name"
}

Response:
{
  "id": "user_id",
  "email": "user@example.com",
  "first_name": "Updated",
  "last_name": "Name"
}

DELETE /api/users/<user_id>

Delete a user.

Response:
{
  "message": "User deleted successfully"
}

Product Endpoints

GET /api/products

Retrieve all products.

Response:
[
  {
    "id": "product_id",
    "name": "Product Name",
    "price": 100,
    "description": "Product Description",
    "images": []
  }

GET /api/products/<product_id>

Retrieve a specific product by ID.

Response:
{
  "id": "product_id",
  "name": "Product Name",
  "price": 100,
  "description": "Product Description",
  "images": []
}

POST /api/products

Create a new product.

Request Body:
{
  "name": "Product Name",
  "price": 100,
  "description": "Product Description",
  "image_url": "path/to/image.jpg"
}

Response:
{
  "id": "new_product_id",
  "name": "Product Name",
  "price": 100,
  "description": "Product Description",
  "images": ["path/to/image.jpg"]
}

PATCH /api/products/<product_id>

Edit specific attributes of a product.

Request Body:
{
  "name": "Updated Product Name",
  "price": 150
}

Response:
{
  "id": "product_id",
  "name": "Updated Product Name",
  "description": "Product Description",
  "price": 150,
  "images": []
}

DELETE /api/products/<product_id>

Delete a product.

Response:
{
  "message": "Product deleted successfully"
}

DELETE /api/products/<product_id>/images/<image_name>

Delete a specific image from a product.

Response:
{
  "message": "Image deleted successfully"
}

Cart Endpoints

GET /api/cart

Retrieve all items in the user's cart.

Response:
[
  {
    "id": "cart_item_id",
    "product_id": "product_id",
    "quantity": 2
  }
]

POST /api/cart

Add a product to the user's cart.

Request Body:
{
  "product_id": "product_id",
  "quantity": 2
}

Response:
{
  "id": "new_cart_item_id",
  "product_id": "product_id",
  "quantity": 2
}

DELETE /api/cart/<product_id>

Remove a product from the user's cart.

Response:
{
  "message": "Product removed from cart"
}

Example cURL Requests

Authentication

curl -X POST http://localhost:5000/api/auth/admin/authenticate \
-H "Content-Type: application/json" \
-d '{
  "email": "admin@example.com",
  "password": "password"
}'

Users

Get all users:

curl -X GET http://localhost:5000/api/users

Get user by ID:

curl -X GET http://localhost:5000/api/users/<user_id>

Create a new user:

curl -X POST http://localhost:5000/api/users \
-H "Content-Type: application/json" \
-d '{
  "email": "newuser@example.com",
  "password": "password",
  "first_name": "First",
  "last_name": "Last"
}'

Update a user:

curl -X PUT http://localhost:5000/api/users/<user_id> \
-H "Content-Type: application/json" \
-d '{
  "first_name": "Updated",
  "last_name": "Name"
}'

Delete a user:

curl -X DELETE http://localhost:5000/api/users/<user_id>

Products

Get all products:

curl -X GET http://localhost:5000/api/products

Get product by ID:

curl -X GET http://localhost:5000/api/products/<product_id>

Create a new product:

curl -X POST http://localhost:5000/api/products \
-H "Content-Type: application/json" \
-H "Authorization: Bearer " \
-d '{
  "name": "Product Name",
  "price": 100,
  "description": "Product Description",
  "image_url": "path/to/image.jpg"
}'

Edit product attributes:

curl -X PATCH http://localhost:5000/api/products/<product_id> \
-H "Content-Type: application/json" \
-H "Authorization: Bearer " \
-d '{
  "name": "Updated Product Name",
  "price": 150
}'

Delete product:

curl -X DELETE http://localhost:5000/api/products/<product_id> \
-H "Authorization: Bearer "

Delete product image:

curl -X DELETE http://localhost:5000/api/products/<product_id>/images/<image_name> \
-H "Authorization: Bearer "
